# ASSET_NOTES.md — Imberion

## Purpose
This file explains how to use the files in this folder.

---

## Folder Role
The `brand_assets/` folder contains the brand, content, and design guidance for the Imberion website.

Before making frontend changes, review the relevant files in this folder.

---

## Files
### `BRAND_GUIDELINES.md`
Use for:
- logo handling
- color palette
- typography
- visual tone
- brand do / don’t

### `DESIGN_SYSTEM.md`
Use for:
- layout decisions
- section patterns
- spacing
- card usage
- CTA style
- motion rules
- hierarchy decisions

### `CONTENT_GUIDE.md`
Use for:
- homepage structure
- tone of voice
- headline clarity
- scannability
- messaging priorities

### Logo files
Use the real logo files when available.  
Do not replace them with placeholders.

---

## Priority Order
When making decisions, follow this order:

1. real assets in this folder
2. brand guidelines
3. design system
4. content guide
5. current live site consistency
6. user instructions in the current session

If there is a conflict, prefer the most explicit and most recent user instruction.

---

## Important Notes
- Do not invent a new visual direction if the brand assets already define one.
- Do not add random components or patterns just because they look modern.
- Keep the site specific to Imberion.
- Prioritize clarity, hierarchy, and brand consistency.
